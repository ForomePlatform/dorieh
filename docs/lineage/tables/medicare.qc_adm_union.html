Overview for medicare.qc_adm_union

Union of data from admissions table with the records discarded because of validation issues

Transformed from medicare.admissions

```sql CREATE view medicare.qc_adm_union AS SELECT reason, year, state, state_iso, admission_year, diagnoses, zip, admission_date, discharge_date, adm_day_of_week, primary_diagnosis, bene_id, bene, pd_hll_hash, icd_hll FROM medicare_audit.admissions UNION ALL SELECT 'OK'::VARCHAR AS reason, year, state, state_iso, admission_year, diagnoses, zip, admission_date, discharge_date, adm_day_of_week, primary_diagnosis, bene_id, bene, pd_hll_hash, icd_hll FROM medicare.admissions ;
