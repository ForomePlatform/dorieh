The gen_dbt_cwl Module

Command line utility to generate a test carcass for a CWL pipeline.

Usage:

python -m dorieh.platform.dbt.gen_dbt_cwl <path/to/cwl/workflow> [<path/to/generated/test>]

If the second argument is omitted, the test is generated in the same directory as original workflow and given a name test_<workflow_name>

Test takes the same parameters as the original workflow plus a list of files containing test cases generated by dorieh.platform.dbt.create_test.py tool.
