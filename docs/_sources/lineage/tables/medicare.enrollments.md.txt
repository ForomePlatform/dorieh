# Table medicare.enrollments

## Overview for medicare.enrollments
Cleaned up and unified Medicare beneficiaries enrollments table,
by state and year. This table is indexed and optimized
for efficient queries


Child table of [medicare.beneficiaries](medicare.beneficiaries.html)

Transformed from [medicare._enrollments](medicare._enrollments.html)

Primary Key: bene_id, year, state
